A powerful Python CLI OSINT tool for performing phone number information lookup using an external API.

The tool provides an animated terminal banner, colorful output, and a clean command-line interface.

⚠️ Educational and cybersecurity research purposes only.

---

✨ Features

• Animated terminal banner
• Colorful CLI output
• API based phone number lookup
• Lightweight Python script
• Easy installation
• JSON formatted results
• Works on Linux / Windows / Termux

📥 Installation

Clone the repository

git clone https://github.com/cybersecuritywithnabhi/osint-number-lookup.git

Open folder

cd osint-number-lookup

Install dependencies

pip install -r requirements.txt

Run tool

python main.py

---

📱 Termux One-Line Install

pkg update && pkg upgrade -y && pkg install python git -y && git clone https://github.com/cybersecuritywithnabhi/osint-number-lookup.git && cd osint-number-lookup && pip install -r requirements.txt && python main.py
